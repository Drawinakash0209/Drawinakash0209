### Things I've built

**Hybrid Explainable AI for Diabetic Retinopathy Detection**
Two-stage diagnostic system using DenseNet-121 and ConvNeXtV2, hitting 99.35% binary screening accuracy. Integrated Grad-CAM for visual explanations, validated by a consulting ophthalmologist.
`PyTorch` `Deep Learning` `Flask`

**Smart Email Assistant**
Full-stack AI assistant with a Chrome extension that generates context-aware email replies with customizable tones directly inside Gmail.
`Spring Boot` `ReactJS` `Gemini API`

**Real-Time Patient Monitoring System**
Serverless event-driven AWS system processing real-time patient telemetry, detecting critical anomalies and triggering sub-second alerts.
`AWS Lambda` `SQS` `DynamoDB` `ReactJS`

**Indoor Sports CRM Platform**
Multi-vendor sports booking platform with dual dashboards — real-time slot availability for players, full booking management for facility owners.
`Laravel` `MySQL` `Tailwind CSS`
